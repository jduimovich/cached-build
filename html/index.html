App Studio Demo Build

Use App Studio Build Pipelines to Build your Container

Be Happy

This application can be built using the App Studio default Build Pipelines

Try

  1. Bootstrap your cluster 
     hack/bootstrap.sh
  2. Create a branch (optional) 
     git checkout -b demo
  3. Install Pipelines 
    ./components/build/hack/install-pipelines.sh
  4. Test Pipelines 
    ./components/build/hack/test-known-build.sh
  5. Run your own container build 
     ./components/build/hack/test-known-build.sh URL
    ./components/build/hack/build.sh  https://github.com/jduimovich/single-container-app
